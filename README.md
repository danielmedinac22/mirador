# Mirador

A place to share AI-generated HTML.

## What's in this repo

- **`alpha/`** — the public alpha (in active development). A Claude Code skill + setup wizard that publishes HTML to your own Vercel. See [`alpha/README.md`](alpha/README.md) for quickstart.
- **`docs/superpowers/specs/`** — design specs:
  - [`2026-05-12-mirador-alpha-design.md`](docs/superpowers/specs/2026-05-12-mirador-alpha-design.md) — alpha (skill-only, BYO Vercel)
  - [`2026-05-12-mirador-design.md`](docs/superpowers/specs/2026-05-12-mirador-design.md) — V1 (managed runtime, multiplayer)
- **`docs/superpowers/plans/`** — implementation plans:
  - [`2026-05-12-mirador-alpha.md`](docs/superpowers/plans/2026-05-12-mirador-alpha.md) — alpha build plan

## Status

The alpha is being built phase by phase. The V1 design exists but no code yet — it's the next milestone after the alpha lands and we have usage signal.
